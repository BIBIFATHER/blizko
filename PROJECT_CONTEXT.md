# PROJECT_CONTEXT.md — Blizko

## Коротко (в 1 абзац)
Blizko — сервис подбора нянь с фокусом на объяснимый мэтчинг, безопасность и стабильность. Ключевая идея: няня = продолжение семьи + добавляет то, чего не хватает. Важны: качество базы → платежи → мобильная упаковка. Принципы: объяснимый AI, безопасность как продукт, человек принимает решение, няни — партнёры.

## Фокус на ближайшее время
1) Качество базы (верификация, OSINT, рекомендации)
2) Платежи/экономика
3) Мобильная упаковка

## Модель совместимости
- Mirror+ (совпадение ценностей/стиля + покрытие потребностей семьи)
- PCM: типы коммуникации (thinker/persister/harmonizer/rebel/imaginer/promoter)
- Объяснимость: почему именно эта няня

## Гарантия прихода
- Soft-hold + приоритетные окна + двойной резерв
- Двойное подтверждение T‑24ч и T‑3–4ч

## Верификация / безопасность
- Документы: OCR/MRZ + face‑match
- Рекомендации: 2 контакта
- OSINT/правовой след: ФССП/суды (hard), форумы (soft)
- Trust‑score: фидбек/отмены

## Текущая реализация в апке (ключевое)
- Календарь: родитель range+grid, няня календарь, админ просмотр
- Экран результата: Mirror/Growth/Stability + эмоц. блоки
- PCM в анкетах + учёт в matching
- Терминология: «потребности семьи» (не «дефицит»)
- Текущий слоган в совместимости: «Blizko+ — подбор, который объясним»
