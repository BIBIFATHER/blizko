# Research Prompt — Blizko Childcare Category

## Назначение

Этот prompt нужен для повторяемого, жёсткого и decision-oriented ресерча по рынку nanny / babysitting / childcare platforms.

## Prompt

```text
Проект: Blizko — trust-first platform для подбора нянь.

Задача:
Сделать фундаментальный market + customer research по категории nanny / babysitting / childcare platforms.

Исследование должно отвечать на 4 вопроса:
1. Какие продукты в этой категории реально работают в разных странах?
2. На чём они зарабатывают, как масштабируются и где ломаются?
3. Какие боли у родителей при поиске няни / childcare?
4. Какие trust-сигналы, workflow и product-решения действительно уменьшают тревогу и повышают конверсию?

Жёсткие правила:
- не пересказывать маркетинговый copy без критики
- каждый важный тезис должен иметь источник
- отделять факт / вывод / рекомендацию
- предпочитать первичные источники:
  - официальные сайты
  - пресс-релизы компаний
  - SEC / regulator filings
  - App Store / product help centers
  - официальные отчёты
- вторичные источники использовать только если нет лучшего первичного подтверждения
- если данных о revenue / profitability / valuation нет публично, прямо писать "не раскрывается публично"
- не придумывать market size, valuation или unit economics
- не делать research theater: в конце должен быть decision memo для Blizko

Обязательные блоки:

1. Market benchmark
- 6-10 компаний
- страна / сегмент / модель / scale
- funding / acquisition / public market cap если есть
- profitability / revenue если публично
- strengths
- weaknesses
- likely success factors
- visible failure modes
- lessons for Blizko

2. Customer pain research
- эмоциональные боли
- trust боли
- operational боли
- financial боли
- hidden jobs-to-be-done
- что пользователи хотят видеть на карточке / в профиле / в flow

3. Product implications for Blizko
- homepage
- parent form
- nanny profile
- trust surfaces
- pricing / payment messaging
- PR and marketing angles

4. Output format
- Executive summary
- Key findings
- Decision implications for Blizko
- Risks / caveats
- Sources

Приоритет мышления:
truth > source quality > usefulness for product decisions > volume
```

## Как использовать

- для глобального benchmark research
- для PR / positioning research
- для copy validation перед редизайном
- для проверки новых claims, прежде чем тащить их на сайт
