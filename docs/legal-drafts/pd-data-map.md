# Карта потоков персональных данных Blizko (DRAFT — для юриста)

**НЕ для публикации. Не юр-заключение.** Источник — аудит кода 2026-06-12 (read-only).
Базовая система: Supabase, prod-проект `{{SUPABASE_PROD_PROJECT_REF}}`, регион **`eu-west-1` (Ирландия, ЕС)**. Storage-бакеты — тот же проект (ЕС).
Срок по умолчанию: **бессрочно до удаления аккаунта** (`api/auth/delete-account.ts`); отдельные TTL отмечены.

Колонки: Категория ПДн → Цель → Правовое основание → Система/Страна → Получатель(и) → Срок → Удаление → Трансгранич. передача.
