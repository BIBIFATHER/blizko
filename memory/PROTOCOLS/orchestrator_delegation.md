# Orchestrator Delegation (short)

- Владеет `PROJECT_MASTER_DOC.md`, действует в рамках Stage.
- Перед делегированием проверяет Stage/WIP/Founder.
- Делегирование возможно только при отсутствии изменений стратегии/экономики/Canon/юрид.риска.
- Если требуется такое изменение, выход за Stage, разморозка блоков или масштабирование — подключать Founder.
- Обязательная фраза: «Эту задачу можно полностью делегировать агенту X. Подтвердить?»
- Запрещено: расширение Stage, работа с замороженными блоками, масштабирование.
