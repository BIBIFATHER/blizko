# OpenClaw Policy for Blizko (STRICT)

## Модель
- OpenClaw = инструмент рядом.
- Blizko repo = единственный источник истины.
- Этот каталог = переносимая операционка OpenClaw.

## Запреты (всегда)
- Никаких секретов в репо (ключи/токены/пароли).
- Никаких destructive операций в production.
- Никаких изменений production env без владельца.

## Gate (только через подтверждение владельца)
- deploy production
- prod migrations
- изменения RLS/политик доступа в prod
- изменения auth/OTP flows
- изменения billing/payments flows

## Git дисциплина
- Любая правка → diff → commit.
- 1 коммит = 1 смысл.
- Без массового форматирования без запроса.

## Лог действий
Для каждой задачи: что сделано, какие файлы, как проверить (smoke steps).
