# Ops Шаблоны (без внешних сервисов)

## 1. Запись в OPS_LOG
```
[YYYY-MM-DD HH:MM] [category] [status]
Кратко: ...
Контекст: ...
Действия: ...
Результат: ...
Файлы: ...
```

## 2. Шаблон чек-листа
```
# Чек-лист: <Название>
Цель: ...
Частота: ... (daily/weekly/on-demand)
Ответственный: ...
Входы: ...
Выходы: ...

Шаги:
1. ...
2. ...
3. ...

Критерий завершения: ...
```

## 3. Шаблон регламента
```
# Регламент: <Название>
Цель: ...
Область применения: ...
Роли: ...
Артефакты: ...
Процесс:
1. ...
2. ...
Отклонения: ...
```

## 4. Шаблон улучшения процесса
```
# Улучшение процесса
Проблема: ...
Текущее состояние: ...
Предложение: ...
Эффект: ...
Риски: ...
Дата внедрения: ...
```

## 5. Шаблон инцидента
```
# Инцидент
Дата/время: ...
Симптом: ...
Влияние: ...
Причина: ...
Временное решение: ...
Фикс: ...
Предотвращение: ...
```
