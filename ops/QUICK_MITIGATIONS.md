# QUICK_MITIGATIONS.md — быстрые меры (без внешних сервисов)

## AI‑matching
- Добавить «must‑have» правила поверх AI‑скора (возраст, опыт, график, гео).
- Порог уверенности: матчи ниже порога — в ручную модерацию.
- Кнопка «почему рекомендовано» + сбор фидбека 1–5.

## Док‑верификация
- Чек‑лист обязательных документов и двойная ручная проверка.
- Флаги verified/unverified + запрет на бронирование без verified.
- Журнал проверок (кто/когда).

## Бронирования
- Стейт‑машина статусов + блокировка слота при подтверждении.
- Идемпотентность (client‑token) для создания брони.
- Суточная сверка целостности бронирований.

## Платежи
- Предавторизация/резерв и явное «paid» состояние.
- Ретрай + ручной инвойс при неуспехе.
- Логирование платежных событий + сверка с заказами.

## Mobile fallback
- Smoke‑тесты ключевых экранов на мобильных.
- Упрощённый lite‑flow (3–4 шага) + автосохранение.
- Контактный fallback (чат/звонок/форма) и ручное создание заявки.
