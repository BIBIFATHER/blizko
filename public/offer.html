Публичная оферта

Blizko · актуальная редакция

1. Общие положения

Настоящая оферта определяет условия использования сервиса Blizko и оказания услуг пользователям.

2. Коммуникации

Пользователь соглашается на получение сервисных уведомлений по SMS, необходимых для оказания услуг, в том числе кодов подтверждения и уведомлений о статусах заявок/заказов. Согласие даётся при регистрации путём установки чекбокса согласия с условиями Политики конфиденциальности/Оферты.

3. Порядок использования

  • Пользователь предоставляет достоверные данные при регистрации.
  • Сервис может запрашивать дополнительные сведения для обеспечения безопасности.
  • Условия оказания услуг могут обновляться; новая редакция публикуется на этой странице.

4. Контакты

По вопросам, связанным с офертой, обращайтесь: hello@blizko.app.
